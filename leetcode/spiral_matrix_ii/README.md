# Spiral Matrix II
https://leetcode.com/problems/spiral-matrix-ii/

<p>
Given a positive integer n, generate an n x n matrix filled with elements from 1 to n2 in spiral order.
</p>

<pre>
<b>Example 1:</b>
Input: n = 3
Output: [[1,2,3],[8,9,4],[7,6,5]]
</pre>

<pre>
<b>Example 1:</b>
Input: n = 1
Output: [[1]]
</pre>
