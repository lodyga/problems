# Palindrome Linked List
https://leetcode.com/problems/palindrome-linked-list/description/

Given the head of a singly linked list, return true if it is a 
palindrome
 or false otherwise.

<b>Example 1:</b>\
Input: head = [1,2,2,1]\
Output: true

<b>Example 2:</b>\
Input: head = [1,2]\
Output: false
