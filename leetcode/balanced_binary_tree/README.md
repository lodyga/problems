# Balanced Binary Tree
https://leetcode.com/problems/balanced-binary-tree/description/

Given a binary tree, determine if it is 
height-balanced

<pre>
Example 1:

  3___
 /    \
9     _20
     /   \
    15    7

Input: root = [3,9,20,null,null,15,7]
Output: true

Example 2:

        __1
       /   \
    __2     2
   /   \
  3     3
 / \
4   4

Input: root = [1,2,2,3,3,null,null,4,4]
Output: false

Example 3:

Input: root = []
Output: true
</pre>
