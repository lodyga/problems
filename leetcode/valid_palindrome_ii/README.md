# Valid Palindrome II
https://leetcode.com/problems/valid-palindrome-ii/

<p>
Given a string s, return true if the s can be palindrome after deleting at most one character from it.
</p>

<pre>
<b>Example 1:</b>
Input: s = "aba"
Output: true
</pre>

<pre>
<b>Example 2:</b>
Input: s = "abca"
Output: true
Explanation: You could delete the character 'c'.
</pre>

<pre>
<b>Example 3:</b>
Input: s = "abc"
Output: false
</pre>
