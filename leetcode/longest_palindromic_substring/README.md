# Longest Palindromic Substring
https://leetcode.com/problems/longest-palindromic-substring/

<p>
Given a string s, return the longest palindromic substring in s.
</p>

<pre>
<b>Example 1:</b>
Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer.
</pre>

<pre>
<b>Example 2:</b>
Input: s = "cbbd"
Output: "bb"
</pre>
