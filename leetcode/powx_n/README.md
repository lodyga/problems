# Pow(x, n)
https://leetcode.com/problems/powx-n/

Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

<b>Example 1:</b>\
Input: x = 2.00000, n = 10\
Output: 1024.00000

<b>Example 2:</b>\
Input: x = 2.10000, n = 3\
Output: 9.26100

<b>Example 3:</b>\
Input: x = 2.00000, n = -2\
Output: 0.25000\
Explanation: 2-2 = 1/22 = 1/4 = 0.25
