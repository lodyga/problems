# Invert Binary Tree
https://leetcode.com/problems/invert-binary-tree/

<p>
Given the root of a binary tree, invert the tree, and return its root.
</p>

<b>Example 1:</b>

<pre>
    __4__             __4__
   /     \           /     \
  2       7    =>   7       2
 / \     / \       / \     / \
1   3   6   9     9   6   3   1
</pre>

Input: root = [4,2,7,1,3,6,9]
Output: [4,7,2,9,6,3,1]

<pre>
<b>Example 2:</b>

  2           2
 / \   =>    / \
1   3       3   1

Input: root = [2,1,3]
Output: [2,3,1]
</pre>

<pre>
<b>Example 3:</b>
Input: root = []
Output: []
</pre>
