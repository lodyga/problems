# Invert Binary Tree
https://leetcode.com/problems/invert-binary-tree/description/

Given the root of a binary tree, invert the tree, and return its root.


<pre>
Example 1:

Input: root = [4,2,7,1,3,6,9]
    __4__
   /     \
  2       7
 / \     / \
1   3   6   9

Output: [4,7,2,9,6,3,1]
    __4__
   /     \
  7       2
 / \     / \
9   6   3   1


Example 2:

Input: root = [2,1,3]
  2
 / \
1   3

Output: [2,3,1]
  2
 / \
3   1

Output: []
