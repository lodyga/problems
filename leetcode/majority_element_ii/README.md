# Majority Element II
https://leetcode.com/problems/majority-element-ii/

<p>
Given an integer array of size n, find all elements that appear more than ⌊ n/3 ⌋ times.
</p>

<pre>
<b>Example 1:</b>
Input: nums = [3,2,3]
Output: [3]
</pre>

<pre>
<b>Example 2:</b>
Input: nums = [1]
Output: [1]
</pre>

<pre>
<b>Example 3:</b>
Input: nums = [1,2]
Output: [1,2]
</pre>
